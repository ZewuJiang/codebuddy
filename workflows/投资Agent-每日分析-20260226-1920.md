## 2.2 美联储反应函数建模（修正泰勒规则）

**泰勒规则（标准版）**: i* = r* + π + 0.5(π - π*) + 0.5(y - y*)

| 参数 | 含义 | 当前值 | 来源 |
|------|------|--------|------|
| r* | 均衡实际利率 | 0.5% | 美联储长期预估 |
| π | Core PCE | 2.6% | 2026年1月数据 |
| π* | 通胀目标 | 2.0% | 美联储法定目标 |
| y - y* | 产出缺口 | +0.3% | 经济韧性 |

> **泰勒规则隐含利率计算**:
> - i* = r* + π + 0.5×(π - π*) + 0.5×(y - y*)
> - i* = 0.5% + 2.6% + 0.5×(0.6%) + 0.5×(0.3%)
> - i* = 0.5% + 2.6% + 0.3% + 0.15% = **3.55%**
>
> 当前联邦基金利率 = 4.25-4.50% → 高于泰勒规则隐含利率约70-95bp → 美联储政策偏紧缩，但考虑§122关税不确定性+地缘风险溢价，选择观望合理
