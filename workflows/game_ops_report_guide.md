# 游戏竞品运营监控 - 使用说明

> 版本: v4.0 | 更新时间: 2026-02-22

## 使用方式

直接对 CodeBuddy 说：

```
超自然行动组 监控周报
```

就会自动完成：采集本周公众号+TapTap+官网动态 → 分类整理 → 采集重点图片 → 生成 PDF + MD 周报。

### 支持的指令格式

| 指令示例 | 说明 |
|---------|------|
| `超自然行动组 监控周报` | 生成本周周报 |
| `三角洲行动 监控周报` | 生成三角洲行动本周周报 |
| `无畏契约 监控周报` | 生成无畏契约端游本周周报 |
| `无畏契约手游 监控周报` | 生成无畏契约手游本周周报 |
| `超自然行动组 第09周监控周报` | 指定周次 |
| `[附公众号截图] 三角洲行动 监控周报` | 附截图辅助识别 |

## 新增游戏

编辑 `game_ops_data/game_registry.json`，添加游戏信息和官方链接即可。

## 产出文件

- `游戏竞品运营监控-{游戏名}-第{周次}周-{日期}.pdf` — MBB 风格 PDF（含图片）
- `游戏竞品运营监控-{游戏名}-第{周次}周-{日期}.md` — Markdown 格式（含图片）
