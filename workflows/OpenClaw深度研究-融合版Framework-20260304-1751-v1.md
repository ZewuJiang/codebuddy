## 四、写作标准

### 格式规范

| 维度 | 标准 |
|------|------|
| **论证结构** | 三层论证法：观点→论据→推演 |
| **核心论点** | 用 Ø 前置，加粗，一句话可读 |
| **数据呈现** | 表格优先，结构化展示 |
| **判断语言** | 直接、判断性，不用"可能""或许"等模糊词 |
| **每段长度** | 3-5行为宜，避免大段文字墙 |
| **引用标注** | 关键数据标注来源（搜索日期+来源） |
| **首尾镜像** | 关键总结与总结一字不差完全镜像 |

### 质量红线

- ❌ 禁止使用AI训练数据作为报告内容的直接来源
- ❌ 禁止模糊数据（"约XXX"→必须给具体数字或标注"截至YYYY-MM-DD"）
- ❌ 禁止无来源的判断（每个核心判断必须有搜索结果支撑）
- ✅ 每段内容三问自查：①来自当期搜索？②时间戳最新？③追问能指向搜索结果？
