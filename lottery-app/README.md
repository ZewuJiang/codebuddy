# 年会抽奖应用

## 功能说明

这是一个精美的年会抽奖应用，具备以下特点：

### 奖品设置
- 🎫 **工卡**：100个
- 🦆 **长鸭**：3个
- ✅ **100% 必中**：所有参与者都能获得奖品

### 使用方法
1. 点击"开始抽奖"按钮，奖品图片开始快速滚动
2. 点击"停止抽奖"按钮，停止滚动并显示中奖结果
3. 系统会自动更新剩余奖品数量
4. 播放庆祝动画

### 技术特点
- 🎨 精美的渐变色UI设计
- ✨ 流畅的动画效果
- 🎊 中奖庆祝特效
- 📊 实时显示剩余奖品数量
- 📱 响应式设计，支持各种屏幕尺寸

### 文件说明
- `index.html` - 主应用文件（包含HTML、CSS、JavaScript）
- `gongka.png` - 工卡奖品图片（需要您提供）
- `change.png` - 长鸭奖品图片（需要您提供）

### 使用步骤
1. 将 `gongka.png` 和 `change.png` 两张图片放入 `lottery-app` 目录
2. 双击打开 `index.html` 文件即可开始使用
3. 或者使用浏览器直接打开该文件

### 抽奖逻辑
- 系统内置103个奖品（100个工卡 + 3个长鸭）
- 每次抽奖从剩余奖品池中随机抽取
- 确保100%中奖，直到所有奖品抽完
- 抽中后自动从奖品池移除，避免重复

## 提示
如果图片无法显示，请确保：
1. 图片文件名正确：`gongka.png` 和 `change.png`
2. 图片文件与 `index.html` 在同一目录下
3. 图片格式为PNG格式

如果暂时没有图片，应用会自动显示奖品名称文字，不影响使用。
