# ✨ Todo List - 待办事项管理应用

一个功能完善、界面精美的待办事项管理应用，帮助您高效组织和追踪日常任务。

## 🎯 功能特性

### 核心功能
- ✅ **任务管理** - 添加、编辑、删除待办事项，标记完成/未完成状态
- 🏷️ **分类系统** - 工作、生活、学习、其他多种分类，快速筛选查看
- 🎯 **优先级设置** - 高/中/低三级优先级，颜色视觉区分（红/橙/绿）
- 📅 **截止日期** - 为任务设置到期时间，过期任务自动高亮提醒
- 💾 **数据持久化** - 使用 localStorage 本地存储，刷新页面数据不丢失
- 🔍 **智能筛选** - 按状态（全部/进行中/已完成）、分类快速过滤

### 高级功能
- 📊 **实时统计** - 显示任务总数和已完成数量
- 📤 **数据导出** - 导出任务数据为 JSON 文件备份
- 📥 **数据导入** - 从备份文件恢复任务数据
- 🗑️ **批量清理** - 一键清除所有已完成任务
- ⌨️ **快捷键支持** - Enter 键快速添加任务

### 用户体验
- 🎨 **简约现代设计** - 清爽的渐变色背景，毛玻璃效果
- 📱 **响应式布局** - 完美适配桌面、平板、移动设备
- ✨ **流畅动画** - 卡片悬停、任务完成、筛选切换动画
- 🎭 **视觉反馈** - 过期任务红色警告，优先级颜色标识

## 🚀 快速开始

### 本地运行

1. **直接打开**
   ```bash
   # 使用浏览器直接打开 index.html 文件
   open index.html  # macOS
   start index.html # Windows
   xdg-open index.html # Linux
   ```

2. **使用本地服务器**（推荐）
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (需要先安装 http-server)
   npx http-server -p 8000
   
   # PHP
   php -S localhost:8000
   ```
   
   然后访问：`http://localhost:8000`

### 部署到服务器

#### 腾讯云 Lighthouse 部署

1. **准备工作**
   - 确保已有轻量应用服务器实例
   - 服务器已安装 Nginx 或 Apache
   - 已安装 TAT（腾讯云自动化助手）

2. **上传文件**
   ```bash
   # 使用 SCP 上传
   scp -r todo-app/* root@your-server-ip:/var/www/html/todo
   
   # 或使用 FTP 工具上传
   ```

3. **配置 Nginx**
   ```nginx
   server {
       listen 80;
       server_name your-domain.com;
       root /var/www/html/todo;
       index index.html;
       
       location / {
           try_files $uri $uri/ /index.html;
       }
   }
   ```

4. **开放端口**
   - 在 Lighthouse 控制台开放 80 端口
   - 或使用防火墙命令：
   ```bash
   ufw allow 80/tcp
   ```

5. **访问应用**
   ```
   http://your-server-ip
   # 或
   http://your-domain.com
   ```

#### 其他部署平台

- **Vercel / Netlify / GitHub Pages**
  - 直接拖拽 `todo-app` 文件夹到平台
  - 或使用 Git 部署

- **云服务器（阿里云、华为云等）**
  - 参考 Lighthouse 部署步骤
  - 配置对应的 Web 服务器

## 📖 使用说明

### 添加任务
1. 在顶部输入框输入任务内容
2. 选择分类（工作/生活/学习/其他）
3. 设置优先级（高/中/低）
4. 可选：设置截止日期
5. 点击"添加"按钮或按 Enter 键

### 管理任务
- **完成任务**：点击任务左侧的圆形复选框
- **编辑任务**：点击任务右侧的 ✏️ 按钮
- **删除任务**：点击任务右侧的 🗑️ 按钮

### 筛选任务
- **按状态筛选**：点击顶部"全部/进行中/已完成"标签
- **按分类筛选**：点击分类过滤按钮
- **组合筛选**：状态和分类可以同时筛选

### 数据管理
- **导出数据**：点击底部"📥 导出数据"按钮，下载 JSON 文件
- **导入数据**：点击"📤 导入数据"，选择之前导出的 JSON 文件
- **清理数据**：点击"🗑️ 清除已完成"，删除所有已完成任务

## 🐛 常见问题

**Q: 数据会丢失吗？**  
A: 数据存储在浏览器本地，除非清除浏览器缓存或卸载浏览器，否则不会丢失。建议定期导出备份。

**Q: 可以多设备同步吗？**  
A: 当前版本不支持云同步。您可以通过导出/导入功能在设备间迁移数据。

**Q: 为什么刷新页面后数据还在？**  
A: 应用使用 localStorage 技术，数据持久化存储在本地。

**Q: 如何清空所有数据？**  
A: 清除浏览器缓存，或在开发者工具中删除 localStorage 中的 `todoAppTasks` 键。
