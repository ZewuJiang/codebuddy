# 四轮审视SOP与估值防错清单

## 估值计算防错清单（每次必检，32项）

| # | 检查项 | 常见错误 | 正确做法 |
|---|--------|---------|---------|
| 1 | **多法加权中枢** | 剔除某一方法后权重不归一化 | 剔除后权重必须重新归一化 |
| 2 | **PE×EPS推导** | 未标明EPS基准 | 标注使用的EPS口径（TTM/Forward/FY20xxE） |
| 3 | **DCF模型方向** | 混淆FCFF和FCFE | FCFF+WACC得EV，需减净债务得权益 |
| 4 | **EV/EBITDA推导** | 忘记减净债务 | EV = EBITDA × 倍数 → 权益 = EV - 净债务 → 每股 |
| 5 | **ROIC均值** | 手动加总出错 | 逐年列出后验算 |
| 6 | **同比增速** | 分子分母取错年份 | 增速 = (本期-上期)/上期 × 100% |
| 7 | **TTM计算** | 季度加总遗漏或重复 | TTM = 最近4个已报告季度之和 |
| 8 | **百分位计算** | 区间取错 | (当前值-下限)/(上限-下限) × 100% |
| 9 | **债务清偿能力** | 混淆净债务vs总债务 | 明确区分两者 |
| 10 | **EPS冲击量化** | 未考虑税率和需求弹性 | 营收冲击→×利润率→×(1-税率)→÷股本 |
| 11 | **ROIC回报倍数** | "每元创造X元回报"表述错误 | 正确：ROIC/WACC = X倍 |
| 12 | **建仓区间一致性** | 正文不同位置给出不同区间 | 全文搜索确保统一 |
| 13 | **⚠️⚠️ 市值排名与竞对市值** | 凭历史印象写排名 | 从companiesmarketcap.com获取+逻辑自洽检查 |
| 14 | **⚠️⚠️ 禁止混用不同时间点数据** | 拼凑排名 | 同一对比同一数据源同一时间点 |
| 15 | **⚠️ 止损位一致性（v4）** | 技术面和投资结论止损不同 | 全文搜索确保统一 |
| 16 | **⚠️ 市场预期合理性（v4）** | 直接用上年同期数据 | 预期值 ≈ 上年同期 × (1+一致增速) |
| 17 | **⚠️ 近似数据统一（v4）** | 同一指标不同位置表述不同 | 全文搜索确保统一 |
| 18 | **⚠️ 图表数据同步（v7）** | 文本和图表用不同季度 | 更新文本后同步更新chart_data |
| 19 | **⚠️ EBITDA推导过程（v7）** | 直接给数字 | 展示：净利润+税+D&A±调整=EBITDA |
| 20 | **⚠️ 货币转换推导（v7）** | 不展示过程 | 展示完整转换链 |
| 21 | **⚠️ DCF简化标注（v7）** | 简化模型未标注 | 标注"简化两阶段示意" |
| 22 | **⚠️ 竞品数据精度（v7）** | 竞品PE写~35x | 使用精确值+数据来源列 |
| 23 | **⚠️ 毛利润增速剪刀差（v8）** | 只写"毛利率下降" | 计算毛利润增速和营收增速，量化差距 |
| 24 | **⚠️ FCF构成质量拆解（v8）** | 只强调正面 | 拆解递延收入/SBC回加/实际运营利润 |
| 25 | **⚠️ Executive Summary双中枢（v8）** | 只给一个中枢 | 加权中枢+保守中枢+当前价相对两者% |
| 26 | **⚠️ Bull Case论据深度（v8）** | 只列标题 | 每条附1-2句论据含数据 |
| 27 | **⚠️⚠️ Non-IFRS调整项口径混淆（v9）** | SBC用错口径 | 从Reconciliation表提取精确值并标注口径 |
| 28 | **⚠️⚠️ D&A从EBITDA反推（v9）** | D&A严重偏差 | 从年报附注三大分项直接提取 |
| 29 | **⚠️⚠️ CapEx口径混用（v9）** | 不同位置不同口径 | 确定统一口径并首次标注 |
| 30 | **⚠️ 估计值残留过多（v9）** | 初稿30+处"~" | 执行五步闭环，最终≤5% |
| 31 | **⚠️⚠️ SOTP估值遗漏（v9）** | 多元化公司只用PE/DCF | SOTP为必备含HoldCo Discount |
| 32 | **⚠️ 历史数据估算（v9）** | 用~号估算 | 从官方年报PDF提取精确值 |

## 量化交叉验证矩阵（⚠️ 撰写后必须逐项验算）

```
┌─────────────────────────────────────────────────────────────────┐
│                    量化交叉验证矩阵                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  █ 基础数据层验证                                               │
│  ├─ PE(TTM) = 股价 / EPS(TTM)                                  │
│  ├─ EPS(TTM) = ΣQ1+Q2+Q3+Q4 EPS（标注稀释调整）               │
│  ├─ PS(TTM) = 市值 / 营收(TTM)                                 │
│  ├─ FCF Yield = FCF(TTM) / 市值                                │
│  ├─ Forward PE = 股价 / Forward EPS（标注来源）                 │
│  └─ 营收(TTM) = ΣQ1+Q2+Q3+Q4 营收                             │
│                                                                 │
│  █ 增速验证层                                                   │
│  ├─ 每个FY的同比增速 = (本期-上期)/上期                        │
│  ├─ TTM同比增速 = (本期TTM - 上年同期TTM) / 上年同期TTM        │
│  └─ EPS同比增速 = (本期EPS - 上期EPS) / 上期EPS               │
│                                                                 │
│  █ 估值计算层验证                                               │
│  ├─ PE法：EPS × PE倍数 = 目标价（三档各验算）                  │
│  ├─ EV/EBITDA法：EBITDA×倍数-净债务 = 权益 → ÷股本 = 每股     │
│  ├─ DCF法：Y1=基年×(1+g), Y2=Y1×(1+g)...逐年验算              │
│  │   ├─ 折现因子：1/(1+WACC)^n 逐年验算                        │
│  │   ├─ 现值：FCF×折现因子 逐年验算                             │
│  │   ├─ Stage1 PV = ΣY1-Y5现值                                 │
│  │   └─ 总权益 = Stage1 PV + Stage2 PV + TV → ÷股本 = 每股     │
│  ├─ FCF Yield法：FCF/Yield = 隐含市值 → ÷股本 = 每股           │
│  └─ 五法加权：Σ(各法中枢×权重) = 综合中枢                     │
│                                                                 │
│  █ 归一化权重验证                                               │
│  ├─ 五法权重之和 = 100%                                         │
│  ├─ 剔除卖方后四法归一化权重之和 = 100%                        │
│  └─ 四法中枢 vs 五法中枢的差值 = 卖方贡献                     │
│                                                                 │
│  █ 风险收益层验证                                               │
│  ├─ 加权期望值 = Σ(目标价×概率)                                │
│  ├─ 隐含上行空间 = (期望值-当前价)/当前价                      │
│  ├─ 超额收益 = 隐含上行空间 - 无风险利率                       │
│  ├─ 上行/下行比 = (乐观-当前) / (当前-悲观)                   │
│  └─ 中枢溢价 = (当前价-估值中枢)/估值中枢                     │
│                                                                 │
│  █ 全文一致性验证                                               │
│  ├─ 搜索"建仓""配置" → 区间一致                               │
│  ├─ 搜索"目标价" → 乐观/中性/悲观一致                         │
│  ├─ 搜索"止损""重新评估" → 止损位一致                         │
│  ├─ 搜索"回购""返还" → 回购金额表述一致                       │
│  ├─ 搜索"仅次于""领先""第一""第二" → 数值↔排名一致           │
│  └─ 搜索"中枢""合理区间" → 估值中枢数字一致                   │
│                                                                 │
│  █ v9新增验算项                                                 │
│  ├─ Non-IFRS调整表验算：Non-IFRS净利润 = IFRS净利润 + Σ调整项 │
│  ├─ D&A三项加总验算：总D&A = 物业折旧+使用权折旧+无形摊销    │
│  ├─ EBITDA验算（使用附注D&A）                                   │
│  ├─ DCF基年FCFF的SBC口径验算                                   │
│  ├─ 毛利验算：毛利 = 营收 × 毛利率（±1%容差）                 │
│  └─ SOTP验算：各板块加总×(1-折价)÷股本=每股SOTP价值            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 四轮审视SOP（v9升级至80+检查项）

### 第一轮：结构完整性检查

- [ ] 10大模块+Executive Summary全部覆盖
- [ ] 每个模块有明确结论/判断
- [ ] 表格格式整齐，关键数字加粗
- [ ] 风险至少8-10个，每个含量化影响+缓释措施
- [ ] 1.5节标题为"盈利质量与现金流"

**v7结构合规检查（全部必备）**：
- [ ] Section 1.2 含机构持仓数据
- [ ] Section 1.3 含PE历史分位数分析
- [ ] Section 2 含"财务表现周期性分析"
- [ ] Section 3.4 含ESG评级+高管薪酬+股权激励
- [ ] Section 3.5 风险表含"缓释措施"列
- [ ] Section 3.6 增速跳跃(>10pp)含解读段落
- [ ] Section 6 含具体日期宏观数据日历
- [ ] Section 7 含布林带+ADX+量价分析+警惕情境
- [ ] Section 8 含触发深度研究信号+期权保护方案
- [ ] Section 9 EBITDA推导+DCF标注+FCF转换+Kelly
- [ ] Section 10 含近8季度逐季附表+趋势总结
- [ ] 竞品估值表精确值+数据来源列

**v8结构合规检查**：
- [ ] Executive Summary双中枢+仓位上限+催化剂≥4+风险≥4
- [ ] Section 1.4 毛利润增速vs营收增速
- [ ] Section 1.5 FCF质量拆解
- [ ] Section 8 毛利率跟踪维度
- [ ] Section 9 Bull Case每条附论据
- [ ] So What段每个核心观点有具体数字
- [ ] 报告末尾核心矛盾一句话提示框

**v9结构合规检查**：
- [ ] Section 1.5或4含Non-IFRS调整项拆解表
- [ ] D&A来源为年报附注（非反推）
- [ ] SBC/CapEx等多口径已标注，全文统一
- [ ] "~"/"(估)"标记≤5%
- [ ] 多元化公司含SOTP估值
- [ ] 历史年度数据来源为官方年报PDF
- [ ] 每个"(估)"附估算方法说明

### 第二轮：数据实时性、一致性与逻辑自洽

- [ ] 数据实时性验证（市值/排名来自实时搜索）
- [ ] 数据时间一致性（同一对比表同一时间截面）
- [ ] 数值vs排名逻辑自洽
- [ ] 文字表述vs数值一致
- [ ] 市场预期合理性验证
- [ ] EPS/PE/营收增速/估值中枢/建仓区间/目标价/止损位一致性
- [ ] 图表与文本数据同步
- [ ] 同比增速官方口径
- [ ] 政策/事件时效性
- [ ] 10Y国债收益率实时性

### 第三轮：计算验证（按量化交叉验证矩阵逐项执行）

- [ ] 基础数据层（PE/PS/FCF Yield/EPS TTM）
- [ ] 增速验证层
- [ ] DCF模型逐年验算+折现因子+现值
- [ ] 多法加权+归一化权重
- [ ] 风险收益比验算
- [ ] v9新增：Non-IFRS调整表/D&A加总/EBITDA/FCFF SBC口径/毛利/SOTP

### 第四轮：表述精确性与全文搜索

- [ ] 净债务vs总债务标明
- [ ] PE触发阈值标明EPS基准
- [ ] 同比增长标明对比时期
- [ ] ROIC使用"ROIC/WACC=X倍"
- [ ] 免责声明日期一致
- [ ] EPS冲击标注间接效应
- [ ] DCF标注FCFF/FCFE
- [ ] 分析师目标价区分表述
- [ ] 估值区间有推导说明
- [ ] 权重分配有说明

**全文搜索关键词验证**：
- "建仓""配置""区间" → 一致
- "目标价""中枢""合理区间" → 一致
- "止损""重新评估" → 一致
- "排名""仅次于""领先""第一" → 数值↔排名一致
- "估""~" → 统计数量≤5%
- "口径" → 多口径指标标注一致
- "SBC""CapEx""D&A""折旧" → 口径一致

## 第四阶段精修检查清单（18项）

1. **论证深度**：每个重大观点有2-3层论据
2. **So What**：每个数据/表格后有解读
3. **前后逻辑链**：Executive Summary与第9章贯通
4. **竞对洞察**：对比表后有3-4条战略洞察
5. **措辞专业度**：避免口语化
6. **数据颗粒度**：保留合理精度
7. **一致性最终扫描**
8. **周期性洞察（v7）**：Section 2含当前周期阶段判断
9. **推导可追溯（v7）**：关键数字有完整推导链
10. **政策时效更新（v7）**
11. **Executive Summary双视角（v8）**：乐观中枢+保守中枢
12. **核心矛盾提炼（v8）**：报告末尾有提示框
13. **Bull Case论证深度（v8）**：每条有数据论据
14. **跟踪指标完备性（v8）**：覆盖核心风险变量
15. **So What量化深度（v8）**：每个核心论点有具体数字
16. **"估"标记最小化审计（v9）**：≤5%，每个残留附估算方法
17. **Non-IFRS/EBITDA推导完整性（v9）**：逐项列出并验算
18. **多口径数据全文一致性（v9）**：搜索确认无混用
