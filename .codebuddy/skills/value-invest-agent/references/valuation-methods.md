## 五法标配 + 市场特色方法

### 方法一：PE估值
- 当前PE vs 历史PE分位（5年/10年）vs 行业中位数 → 保守/基准/乐观三档
- ⚠️ 每档标注EPS口径（TTM/FY20xxE/Forward）
- ⚠️ A股注意：区分扣非PE和含非经常性损益PE

### 方法二：EV/EBITDA
- EV = EBITDA × 倍数 → 权益 = EV - 净债务 → 每股 = 权益/股本
- ⚠️ 完整推导链必须展示
- ⚠️ EBITDA推导必须展示：净利润 + 税 + D&A ± 其他调整 = EBITDA
- ⚠️ D&A必须使用年报附注值（铁律#36）
- ⚠️ 多币种注意：全部使用本币，跨市场标注汇率

### 方法三：DCF估值
- 自由现金流折现（含逐年折现表+5×5敏感性矩阵）
- ⚠️ 必须注明FCFF vs FCFE处理方式
- ⚠️ 核心参数表：基年FCF、Stage1增长率、Stage2增长率、永续增长率、WACC、β值
- ⚠️ WACC因市场而异（无风险利率按该市场国债收益率）
- ⚠️ DCF基年FCFF = OCF - CapEx - SBC，确认SBC使用Non-IFRS调整口径（铁律#35）
- ⚠️ 若使用简化两阶段模型，标注"简化两阶段示意"
- ⚠️ FCF货币转换必须展示完整计算链

### 方法四：FCF Yield
- 合理FCF Yield区间 → 隐含市值 → 隐含股价
- ⚠️ 该方法未考虑增长溢价，对高增长公司可能偏保守

### 方法五：卖方一致预期
- 分析师目标价均值（市场情绪参考，权重较低）
- ⚠️ 表述规范：区分"一致均值"和"个别分析师目标价区间"
- 含"卖方共识偏差提示"

### 可选方法
- **PEG**（高增长公司）
- **PS**（亏损公司）
- **PB**（金融/重资产公司；A股PB-ROE法）
- **EV/EBIT**（日股特色）
- **PBR**（日本低PBR改革关注度高）

### SOTP估值（⚠️ 多元化控股公司必备 — 铁律#40）

持有3+独立业务板块或大量投资组合的公司，SOTP为**必备**。须含：
1. **各业务板块独立估值**（用行业适配倍数）
2. **投资组合估值**（上市投资→实时市值×持股比例；未上市→可比公司估值法）
3. **HoldCo Discount**（控股公司折价10-25%，标注理由）
4. **汇总表格式**：

```markdown
| 板块 | 估值方法 | 独立估值 | 权益占比 | 贡献 |
|------|---------|---------|---------|------|
| {业务1} | PE {X}x | {A}亿 | 100% | {A}亿 |
| {业务2} | PS {X}x | {B}亿 | 100% | {B}亿 |
| 上市投资组合 | 市值法 | {C}亿 | 按持股比例 | {C'}亿 |
| 未上市投资 | 可比公司法 | {D}亿 | — | {D}亿 |
| **合计** | | | | **{总}亿** |
| (-) HoldCo折价 | | {X}% | | **-{折}亿** |
| **SOTP每股价值** | | | | **{每股}** |
```

## 风险分析框架（含EPS冲击量化）

每个风险必须包含：
1. **核心问题描述**
2. **风险敞口**（涉及的营收/利润规模）
3. **影响程度**（对EPS/股价的量化冲击）
4. **发生概率**（高/中/低+定性依据）
5. **缓释措施**（公司已有的应对方案）
6. **EPS冲击公式**：营收影响 × 利润率 × (1-税率) ÷ 股本 = EPS影响

## 护城河评估（6维度）

1. **品牌力**: 品牌溢价能力、客户忠诚度
2. **网络效应**: 用户越多价值越大、生态锁定
3. **成本优势**: 规模经济、学习曲线
4. **转换成本**: 迁移成本、数据绑定
5. **技术壁垒**: 专利/IP、研发壁垒
6. **牌照/特许权**: 行业准入壁垒

> 每维度1-10分，总分加权后给出等级：极宽(Wide Moat)/宽(Moat)/中等(Narrow)/窄(None)
> 必须给出未来3-5年"扩宽/持平/收窄"判断及理由
