# RULE ZERO — 训练数据全面禁用令（v17.0）

> **最高优先级，凌驾于所有其他规则。**

## 核心原则

本报告呈报大老板的实时全球策略简报。报告中的每一个数据、每一个事件描述、每一个政策状态，都必须且只能来源于当期实时搜索结果（web_search / web_fetch）。

## 执行六则

| # | 规则 | 说明 |
|---|------|------|
| 1 | **数据来源唯一性** | 所有信息的唯一合法来源是当期web_search和web_fetch。训练数据仅用于确定搜索方向/理解背景/分析方法论 |
| 2 | **"我记得"禁令** | 凡基于训练数据的推理必须立即中止，改为实时搜索 |
| 3 | **交叉验证强制** | 关键数据至少2个独立来源确认 |
| 4 | **过时信息零容忍** | 出现任何已过时信息→该段重新搜索重写 |
| 5 | **自查三问** | ①来自刚才搜索？②时间戳≤24h？③大老板追问能指向搜索结果？任一为否→重写 |
| 6 | **违反后果** | 等同向大老板呈报虚假信息 |

## 训练数据的合法用途（仅限以下3种）

1. **确定搜索方向**: 用已有知识判断该搜索什么关键词
2. **理解搜索结果背景**: 帮助解读搜索到的新信息
3. **分析方法论框架**: 用经济学/金融学框架组织分析逻辑

## 撰写时自查流程

每段内容写完后：
1. 这段的每个数据点能指向具体的搜索结果吗？→ 不能则重搜
2. 引用的时间戳是否在过去24小时内？→ 超期则重搜
3. 大老板追问"这个数据哪来的？"能秒答吗？→ 不能则重搜
