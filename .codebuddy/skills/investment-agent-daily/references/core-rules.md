# v17.3 核心规则（18条）

> 从旧框架80条铁律精简而来，v17.3新增3条数据完整性和去重规则（来源于0303实战教训）。

## 数据规则（8条）

| # | 规则 | 说明 | 违反后果 |
|---|------|------|---------|
| 1 | **RULE ZERO** | 所有数据只能来自当期实时搜索 | 致命——全段重写 |
| 2 | **数据实时性** | 每个数据必须过去24h内最新，标注数据时间 | 致命——重新搜索 |
| 3 | **交叉验证** | 关键数据（指数/个股/汇率）至少2源确认 | 高风险——补充验证 |
| 4 | **政策实时校验** | 政策事件必须搜索确认最新状态，严禁依赖记忆 | 致命——重新搜索 |
| 5 | **涨跌幅计算** | 每个涨跌幅必须公式计算`(现价-前收)/前收*100%`，禁止凭直觉 | 高风险——重新计算 |
| **6** | **数据完整性扫描（v17.3）** | 撰写前必须对照必填数据清单逐项确认，M7×7+GICS×11+指数×3+VIX+焦点≥2+亚太×4+大宗×6全部有精确值 | **致命——禁止进入撰写阶段** |
| **7** | **零空位零模糊（v17.3）** | 表格严禁"—""-""N/A""~""约""小幅""略"等。搜不到→换源继续搜→实在无法→删除该行 | **致命——报告返工** |
| **8** | **Google Finance SOP（v17.3）** | 美股个股/ETF统一通过`web_fetch("google.com/finance/quote/{TICKER}:{EXCHANGE}")`批量采集，确保前收+现价精确值 | 高风险——数据不精确 |

## 内容规则（5条）

| # | 规则 | 说明 | 违反后果 |
|---|------|------|---------|
| 9 | **有事才说** | AI产业链/关键标的无动态时不写，有动态时说清楚 | 中风险——精简冗余 |
| 10 | **建议可执行** | 操作建议必须具体（做什么/什么价位/什么时候） | 致命——重写建议 |
| 11 | **结论先行** | 每段先给结论再给证据 | 中风险——调整结构 |
| 12 | **200-300行** | 报告总行数控制在200-300行，阅读时间约5分钟 | 中风险——精简或补充 |
| **13** | **段落去重（v17.3）** | 同一事件只在最相关段落出现一次：§3写个股/公司事件，§4写机构/大师动向，不交叉重复 | **高风险——删除重复段落** |

## 流程规则（5条）

| # | 规则 | 说明 | 违反后果 |
|---|------|------|---------|
| 14 | **交易日检测** | 生成前执行`date "+%A"`确认星期几，周末自动切换模板 | 致命——模板错误 |
| 15 | **PDF双输出** | MD + PDF同步输出，PDF使用`md_to_pdf.py`两轮渲染法 | 中风险——补充PDF |
| 16 | **文件命名** | `{HHMM}`取报告实际开始时刻系统时间，严禁凑整 | 高风险——重新命名 |
| 17 | **端到端零堵点** | 遇到数据源异常自动降级，不停下等用户 | 中风险——手动切源 |
| 18 | **禁用词零容忍** | 关注/注意/适时/伺机/择机/酌情/留意/跟踪 → 替换为具体建议 | 高风险——逐一替换 |

## 禁用词替换对照

| 禁用词 | 替换为 |
|--------|--------|
| 关注XX | "当XX达到$YYY时考虑买入" |
| 注意风险 | "下行风险XX%，建议止损位$YYY" |
| 适时/伺机/择机 | "在MM/DD前完成" 或 "当XX条件满足时" |
| 酌情 | "[具体数量/比例]" |
| 留意/跟踪 | "[具体触发条件]+[具体动作]" |
| 小幅/略涨/约 | "[精确数值到小数点后两位]" |

## 数据禁用表述对照（v17.3新增）

| 禁用表述 | 替换为 | 示例 |
|----------|--------|------|
| "—" / "-" / "N/A" | 精确数值 | "—" → "$182.37" |
| "~$264" / "约$264" | "$264.72" | 去掉波浪号和"约" |
| "小幅上涨" | "+0.20%" | 精确涨跌% |
| "略有下跌" | "-1.68%" | 精确涨跌% |
| "VIX回落至22-23" | "VIX收21.44 (+7.96%)" | 精确值+精确涨跌% |
| "MSFT ~$407" | "MSFT $398.55 (+1.48%)" | Google Finance精确值 |
