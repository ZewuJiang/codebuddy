## 常见错误快速修复

| 错误 | 修复方法 |
|------|---------|
| Google Finance 403 | web_search → 东方财富/金投网 |
| 年份混淆（搜到2025数据） | 以Google Finance为锚验证当前年份 |
| VIX方向写反 | VIX↑=恐慌↑=🔴，VIX↓=恐慌↓=🟢 |
| 北向资金买入但标"看好" | 检查内资派发：外资买+指数不涨=内资卖 |
| PDF中文乱码 | 检查STHeiti是否在font-family首位 |
| 搜索无结果 | 扩大关键词+换中/英文+换数据源 |
